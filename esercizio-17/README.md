# Esercizio

Alla cassa del supermercato, il primo cliente della fila viene servito e se ne va; rimuovilo della coda per far avanzare gli altri.


## Ragionamento

- Dichiaro l'array che rappresenta la coda dei clienti alla cassa del supermercato
- Utilizzo il metodo .shift() per rimuovere il primo cliente in cima alla lista
   - stampa l'array aggiornato mostrando chi è rimasto in coda
