# Esercizio

- Stai usando un browser web, clicchi il pulsante "Indietro" che ti riporta alla pagina precedente; rimuovi l'ultima URL visitata dallo storico di navigazione

## Ragionamento

- Dichiaro l'array con le URL visitate nello storico
- Utilizzo il metodo .pop() per rimuovere l'ultimo elemento caricato (l'ultima URL)
    - stampa l'array aggiornato dello storico di navigazione
