# Esercizio

Hai registrato le tue spese giornaliere in ordine cronologico; invertilo per vedere l'ultima spesa effettuata in cima alla lista.

## Ragionamento

- Dichiaro l'array contenente i nickname dei contatti bloccati
- Definisco il nickname dell'amico da controllare
- Utilizzo il metodo .includes() per verificare se il nickname è presente nell'array dei bloccati
- SE il metodo restituisce vero
   - stampa un avviso dicendo che l'utente è bloccato
- ALTRIMENTI
   - stampa un avviso di successo
