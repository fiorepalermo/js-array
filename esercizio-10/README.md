# Esercizio

- Devi generare un codice CSV da inviare al server; trasforma un array in una stringa con i valori separati da ","


## Ragionamento

- Dichiaro l'array contenente i dati da trasmettere
- Trasformo l'array in una stringa usando il metodo .join(",") separando i valori con una virgola
    - stampa la stringa CSV risultante
