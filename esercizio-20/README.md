# Esercizio

In un'app di ricette, hai gli ingredienti in un array; stampali in modo leggibile nella descrizione del piatto.

## Ragionamento

- Dichiaro l'array con la lista degli ingredienti della ricetta
- Unisco tutti gli elementi dell'array all'interno di una stringa leggibile usando il metodo .join(", ")
- stampa la descrizione del piatto ottenuta
