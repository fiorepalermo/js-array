# Esercizio

- In un parcheggio numerato, l'utente cerca la sua auto; trova la posizione (indice) del suo numero di targa


## Ragionamento

- Dichiaro l'array delle targhe nel parcheggio
- Dichiaro la targa cercata dall'utente
- Uso il metodo .indexOf() passando la targa cercata per trovare l'indice corretto
    - stampa l'indice trovato
