# Esercizio
Data la lista VIP per una serata, verifica se l'utente X è presente nella lista VIP o se è nella lista standard

## Ragionamento

- Dichiaro l'array con la lista dei nomi VIP
- Dichiaro una variabile con il nome dell'utente da verificare
- Dichiaro una variabile booleana trovato impostata su falso
- PER ogni nome nella lista VIP
- SE il nome corrente è uguale al nome dell'utente
    - imposto trovato su vero
    - interrompo il ciclo
- SE trovato è vero
    - stampa l'utente è nella lista VIP
- ALTRIMENTI
    - stampa l'utente è nella lista standard
