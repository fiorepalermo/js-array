# Esercizio

Stai chattando con un tuo amico; quando arriva un nuovo messaggio, aggiungilo all'intera conversazione

## Ragionamento

- Dichiaro l'array contenente i messaggi della conversazione della chat
- Definisco una variabile contenente il nuovo messaggio in arrivo
- Utilizzo il metodo .push() per inserire il messaggio alla fine dell'array della chat
   - stampa l'array della conversazione aggiornato
