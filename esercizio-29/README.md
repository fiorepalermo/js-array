# Esercizio

Devi inserire una pubblicità dopo la terza notizia nel feed di un giornale online; inserisci l'elemento pubblicitario nella posizione corretta senza sovrascrivere nulla.

## Ragionamento

- Dichiaro l'array contenente la lista di notizie presenti nel feed del giornale
- Definisco il contenuto pubblicitario da inserire nel testo
- Utilizzo il metodo .splice() impostando come indice di partenza la posizione 3 (dopo la terza notizia), rimuovendo 0 elementi e inserendo la stringa pubblicitaria
   - stampa l'array del feed del giornale aggiornato
