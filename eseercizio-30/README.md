# Esercizio

Dopo una partita di tris, inizializza la griglia di gioco con il valore "vuoto" a tutte le celle.

## Ragionamento

- Dichiaro un nuovo array impostando la grandezza totale pari alle 9 celle del gioco del tris
- Riempio uniformemente tutte le caselle dell'array appena creato con il valore "vuoto" tramite il metodo .fill()
   - stampa l'array della griglia di gioco iniziale
