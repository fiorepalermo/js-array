# Esercizio

In un magazzino, gli ultimi scatoloni arrivati sono i primi ad essere spediti. Il magazziniere spedisce un pacco, stampa la lista aggiornata

## Ragionamento

- Dichiaro l'array degli scatoloni pronti all'interno del magazzino
- Rimuovo l'ultimo scatolone arrivato (l'ultimo inserito in coda) tramite il metodo .pop()
    - stampa l'array del magazzino aggiornato mostrando i pacchi rimanenti
