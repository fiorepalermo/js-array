# Esercizio

Devi inizializzare il report mensile per il mese successivo; imposta temporaneamente tutti i valori a "N/D".

## Ragionamento

- Dichiaro un nuovo array vuoto specificando la dimensione richiesta per i mesi
- Sovrascrivo tutti gli elementi interni dell'array impostando il valore "N/D" usando il metodo .fill()
   - stampa l'array del report mensile inizializzato
