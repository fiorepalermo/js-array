# Esercizio

Nella classifica di Serie A, prendi le squadre che giocheranno l'Europa League (dal 5° all'8° posto in classifica)


## Ragionamento

- Dichiaro l'array contenente le squadre di calcio della classifica di Serie A
- Estraggo la porzione di array che va dalla 5ª alla 8ª posizione usando il metodo .slice(4, 8)
   - stampa il sotto-array ottenuto con le squadre qualificate in Europa League
