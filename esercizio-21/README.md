# Esercizio

Gestisci una fila di pazienti in sala d'attesa; quando il dottore chiama il primo paziente, rimuovilo dalla lista.

## Ragionamento

- Dichiaro l'array contenente i nomi dei pazienti in lista d'attesa
- Utilizzo il metodo .shift() per rimuovere il primo paziente in testa alla lista
   - stampa la lista dei pazienti rimasti ad aspettare
