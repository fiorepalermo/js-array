# Esercizio

- Dato un array di prodotti e uno di quantità, stampa se il prodotto è esaurito o se le scorte sono basse

## Ragionamento

- Dichiaro l'array dei nomi (ingredienti1)
- Dichiaro l'array delle quantità
   - PER ogni quantità
- SE la quantità è uguale a 0
    - stampa "ESAURITO" insieme al nome dell'ingrediente
- ALTRIMENTI SE la quantità è minore di 5
      - stampa "Scorte basse" insieme al nome dell'ingrediente
