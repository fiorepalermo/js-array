# Esercizio

Sei il buttafuori di un locale VIP: arriva un ospite importante che deve saltare la fila; inserisci il suo nome all'inizio della lista d'ingresso.

## Ragionamento

- Dichiaro l'array contenente la lista d'ingresso del locale
- Definisco il nome dell'ospite importante (VIP)
- Utilizzo il metodo .unshift() per aggiungere il nome dell'ospite VIP all'inizio dell'array
    - stampa l'array della lista d'ingresso aggiornato
