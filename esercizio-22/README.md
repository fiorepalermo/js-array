# Esercizio

Trova la posizione del giorno della settimana in cui è caduto il tuo compleanno quest'anno

## Ragionamento

- Dichiaro l'array con i giorni della settimana ordinati
- Definisco il giorno preciso in cui cade il compleanno
- Ricerco la posizione numerica del giorno scelto nell'array utilizzando il metodo .indexOf()
- stampa l'indice numerico ottenuto
