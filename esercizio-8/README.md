# Esercizio

- Un ospedale ha 7 infermieri con questi turni: ["mattina", "pomeriggio", "notte", "mattina", "pomeriggio", "notte", "mattina"]. Conta quanti infermieri lavorano in ciascun turno. 


## Ragionamento

- Dichiaro l'array con i turni degli infermieri
- Dichiaro tre variabili contatore (mattina, pomeriggio, notte) inizializzate a 0
- PER ogni turno presente nell'array
- SE il turno attuale è uguale a "mattina" -> incremento il contatore mattina
- ALTRIMENTI SE il turno attuale è uguale a "pomeriggio" -> incremento il contatore pomeriggio
- ALTRIMENTI SE il turno attuale è uguale a "notte" -> incremento il contatore notte
   - stampa il risultato finale dei tre contatori
