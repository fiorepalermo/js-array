# Esercizio

In un menu digitale, verifica se un ingrediente specifico è presente nella lista degli allergeni di un piatto.

## Ragionamento

- Dichiaro l'array con l'elenco degli allergeni presenti all'interno di un piatto scelto
- Definisco la stringa contenente l'ingrediente da monitorare
- Utilizzo il metodo .includes() per verificare se l'ingrediente inserito fa parte dell'elenco allergeni
- SE il risultato del metodo è vero
   - stampa un messaggio di allarme per la presenza dell'allergene
- ALTRIMENTI
   - stampa un messaggio di conferma di sicurezza
