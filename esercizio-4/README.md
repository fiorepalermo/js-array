# Esercizio
- Date le temperature settimanali, trova il valore massimo, il minimo e calcola la media aritmetica delle sette giornate

## Ragionamento

- Dichiaro l'array delle temperature settimanali
- Dichiaro la variabile max uguale alla prima temperatura
- Dichiaro la variabile min uguale alla prima temperatura
- Dichiaro la variabile somma inizializzata a 0
- PER ogni temperatura
- SE la temperatura corrente è maggiore di max
   - assegno a max il valore della temperatura corrente
- SE la temperatura corrente è minore di min
   - assegno a min il valore della temperatura corrente
   - aggiungo la temperatura corrente alla variabile somma
- Calcolo la media dividendo somma per la lunghezza dell'array
   - stampa i valori di max, min e media
