# Esercizio

Stai creando una funzione "Undo" (Annulla); inverti l'ordine delle azioni compiute per poterle annullare nell'ordine corretto

## Ragionamento

- Dichiaro l'array contenente la sequenza temporale delle azioni eseguite dall'utente
- Inverto completamente la disposizione dell'array tramite il metodo .reverse() in modo da posizionare le ultime azioni per prime
    - stampa l'array ottenuto pronto per la logica di annullamento
