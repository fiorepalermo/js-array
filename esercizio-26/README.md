# Esercizio

Hai una lista di priorità lavorative; il capo ti assegna un'emergenza da fare subito: inseriscila come primo elemento dell'array e stampa la lista.

## Ragionamento

- Dichiaro l'array contenente la lista ordinata delle attività lavorative
- Definisco l'attività di emergenza assegnata dal capo
- Utilizzo il metodo .unshift() per posizionare l'emergenza in cima all'array come primo elemento assoluto
   - stampa l'array aggiornato delle priorità lavorative
