# Esercizio

Stai sviluppando una "To-Do List": dato un array di task già presenti, chiedi all'utente di aggiungere una nuova attività.

## Ragionamento

- Dichiaro l'array contenente le attività (task) già presenti nella lista
- Mostro un input a schermo tramite prompt() chiedendo all'utente una nuova attività
- Utilizzo il metodo .push() per inserire la risposta dell'utente alla fine dell'array dei task
   - stampa l'array della To-Do List aggiornato
